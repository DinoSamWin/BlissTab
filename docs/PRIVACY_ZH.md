# StartlyTab 隐私与数据处理披露政策

**版本：2026.1 (完全符合 Chrome 网上应用店合规要求)**

本政策旨在详细说明 StartlyTab 如何收集、处理、存储、共享用户数据，并明确您的数据删除权利。

## 1. 详细的数据收集 (Data Collection)

StartlyTab 仅收集实现核心功能所必需的以下数据：

*   **个人身份与账户**：Google 账户 ID、电子邮件地址、用户名称及头像。这些数据作为多端同步的唯一标识。
*   **面板自定义对象**：保存的网站 URL、自定义标题、用户上传的链接图标（存储在 Supabase Storage）。
*   **财务与会员记录**：订阅方案（Free/Pro）、兑换码使用历史、交易参考 ID（用于会员资格校验）。
*   **环境上下文 (Telemetry)**：应用心跳时间、标签页数量、多媒体播放/静音状态、语言与主题偏好。
*   **心理记录 (私密数据)**：心情日志分值、个人笔记与待办清单。

## 2. 数据处理目的 (Data Processing)

我们对上述数据进行处理的唯一目的如下：

*   **跨设备同步 (Cross-Sync)**：利用账户 ID 在不同浏览器实例间同步您的面板布局和链接。
*   **AI 共鸣校准**：分析“环境上下文”（如当前是否开启了大量标签页）来调整 AI 陪伴语的语气和同理心强度。
*   **会员资格校验**：处理兑换码和订阅计划，以开放高级功能权限。
*   **本地趋势分析**：基于“心理记录”在本地分析情绪波动，识别如“深夜工作周期”等模式并提供支持。

## 3. 数据存储、安全与保留 (Storage & Retention)

*   **存储位置**：
    *   **云端 (Supabase)**：面板布局、链接、账户身份和设置信息存储在基于 Google Cloud 架构的 Supabase 数据库中，使用 AES-256 静止加密。
    *   **本地 (IndexedDB)**：敏感的心情记录、笔记存储在您个人的硬件设备上。
*   **安全性**：所有数据传输均通过 TLS 1.2+ 加密。云端访问受到严格的 RLS (行级安全) 策略保护。
*   **保留期限**：
    *   本地活动缓存和 AI 生成日志每 **30 天** 自动清除。
    *   账号数据将保留至用户主动注销或发起删除请求。

## 4. 数据共享与第三方传输 (Sharing & Disclosure)

**严正声明：我们不会将您的数据出售给广告商、数据经纪商或任何第三方营销机构。**

数据仅在以下必要情况下通过加密通道传输给核心基础设施伙伴：
*   **Google / Supabase**：提供云端数据库存储。
*   **Vercel**：提供应用托管服务。

## 5. 数据删除与用户权利 (User Rights)

您对自己的数据拥有绝对控制权：
*   **本地删除**：您随时可以通过应用设置清除本地缓存和所有私密记录。
*   **云端销毁**：如需彻底删除存储在 Supabase 云端的账号数据，请发送邮件至 [support@startlytab.com](mailto:support@startlytab.com)。我们将在 7 个工作日内完成物理层面的数据抹除。
