# StartlyTab 隐私政策 (合规更新版)

**最后更新日期：2026年4月**

StartlyTab ("我们") 致力于保护您的隐私。本政策详细说明了我们在为您提供服务时如何收集、处理、存储和共享您的数据，以完全符合 Chrome 网上应用店开发者计划政策。

## 1. 用户数据收集披露

我们收集以下类型的数据以提供核心产品功能：

*   **身份信息 (Identifiable Information)**：Google 账号 ID、电子邮件地址和头像。
    *   **用途**：用于账号身份识别、跨设备同步和订阅状态验证。
*   **位置与环境上下文 (Contextual Data)**：基于 IP 的大致区域位置、当地时间、电池电量。
    *   **用途**：用于本地化搜索/天气显示，并为 AI 提供环境感知背景。
*   **数字环境感知 (Digital Environment)**：打开的标签页数量、音频播放状态、闲置时间、下载状态。
    *   **用途**：让 AI 能够感知您的“数字步调”（例如：是否正在忙碌或长时间闲置），从而提供更人性化的反馈。
*   **用户生成内容与情绪数据 (Personal Records)**：心情记录（情绪值）、心理健康模式（如检测到“深夜加班”）、个人笔记、待办事项。
    *   **用途**：核心功能，用于生成心理健康报告、基准分析和个性化陪伴。
*   **浏览活动与智能缓存 (Activity & Cache)**：浏览历史快照（用于快速访问功能）、AI 生成的内容缓存。
    *   **用途**：驱动“快速搜索”功能，并利用“本地智能缓存”存储 30 天内的生成内容以降低延迟。

## 2. 数据处理原则

*   **本地优先处理**：大多数敏感数据（如浏览历史分析、心情模式推断）均在您的设备本地完成处理。
*   **AI 个性化**：在您发起 AI 交互时，我们会发送**匿名化**的环境背景（如“当前打开了 5 个标签页”）给大模型，使回复更具共鸣。
*   **禁止模型训练**：发送给 AI 功能的数据绝不会用于训练公开的机器学习模型，并在会话结束后立即清除。

## 3. 数据存储与保留

*   **本地存储**：绝大多数敏感数据存储在您的浏览器本地（IndexedDB 和 chrome.storage.local）。
*   **保留期限**：活动日志和生成缓存会在 **30 天** 后自动清除。账号设置数据在您的账号激活期间保留。您随时可以点击“擦除所有数据”彻底清空本地存储。

## 4. 数据共享与传输

*   **零共享承诺**：我们**绝不**将您的个人记录、笔记或浏览历史出售或转让给第三方广告平台或数据经纪商。
*   **必要传输**：数据传输仅限于必要的基础设施合作伙伴（如 Google Firebase/Supabase, Vercel），且仅用于核心应用运行和安全云同步。

## 5. 权限使用说明

*   `bookmarks`: 用于在侧边栏显示和管理书签。
*   `history`: 用于从命令栏快速搜索历史记录，以及 AI 上下文参考。
*   `idle` / `tabs`: 让 AI 能够感知您的活动水平（如长期不在焦点）。
*   `storage`: 用于在本地保存心理模式、笔记和应用状态。
*   `identity`: 管理 Google 安全登录及同步。

## 6. 有限使用 (Limited Use) 合规声明

我们对从 Google API 获得的任何信息的使用均严格遵守 [Chrome Web Store User Data Policy](https://developer.chrome.com/docs/webstore/program-policies/limited-use)，包括“有限使用”限制。我们保证：
*   不允许任何人员阅读用户数据（除非获得技术支持的明确许可）。
*   不得将数据用于信用评估或定向广告。
