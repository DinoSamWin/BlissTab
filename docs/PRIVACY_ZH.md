# StartlyTab 隐私政策 (深度透明版)

**最后更新日期：2026年4月**

为了确保 100% 的透明度并符合 Chrome 网上应用店的最严苛审核要求，我们在此详细列出 StartlyTab 收集的所有数据项及其用途。

## 1. 详细数据收集披露

我们收集以下数据以实现多端同步和个性化体验：

### 1.1 账号与身份 (Identity)
*   **Google 账号 ID**：作为账号唯一识别码。
*   **电子邮件地址**：用于身份验证、订阅确认及订单找回。
*   **个人资料名称与头像**：用于应用内的个性化欢迎展示。

### 1.2 自定义快捷链接 (Gateways)
*   **网站 URL**：您保存的快捷访问地址。
*   **自定义标题**：您为网站设置的个性化显示名称。
*   **自定义图标**：您上传或指定的网站图标（存储在 Supabase Storage 中）。
*   **布局与分组**：您对图标的分组、排序和文件夹管理信息。

### 1.3 订阅、订单与兑换 (Commerce)
*   **订阅状态**：当前是 Free 还是 Pro 计划。
*   **兑换码记录**：您曾经使用过的兑换码及其使用时间。
*   **会员有效期**：计划的到期和续费时间戳。
*   **订单锚点**：用于关联支付平台的交易 ID。

### 1.4 回访与会话元数据 (Telemetry)
*   **最后活跃时间/心跳 (Heartbeat)**：通过检测心跳记录您的回访频率。
*   **页面停留与返回**：用于计算 48 小时情绪基准。
*   **应用设置**：界面语言选择、深色/浅色模式主题、AI 陪伴人格设置。
*   **钉选内容 (Pinned Snippet)**：记录您最后收藏或喜欢的 AI 金句。

### 1.5 数字环境感知 (Environment)
*   **标签页统计**：当前浏览器打开的标签页数量。
*   **媒体状态**：是否有音频正在播放、是否静音、是否有文件正在下载。

### 1.6 心理记录与情绪 (Personal Records)
*   **心情日志**：您主动记录的情绪分值与感受。
*   **笔记与清单**：您在应用内记录的文字内容。
*   **分析模式**：根据记录推断的“深夜加班”或“能量低谷”等心理状态模式。

## 2. 数据处理目的

*   **同步服务**：确保您在不同电脑上打开 StartlyTab 时，快捷链接、分组和设置完全一致。
*   **会员权益**：验证您的订阅状态或兑换码，以开放 Pro 功能。
*   **AI 共鸣**：根据您的回访时间、忙碌程度（标签页数量）和心情，调整 AI 生成的内容，使其更具陪伴感。

## 3. 存储与保留

*   **云端存储 (Supabase)**：账号、设置、链接和订单信息存储在加密的 Supabase 云数据库中。
*   **本地存储 (IndexedDB)**：心情记录、个人笔记等高度敏感数据优先存储在您本地硬盘上。
*   **自动销毁**：活动日志和 AI 生成缓存将在 **30 天** 后自动从本地清除。

## 4. 有限使用 (Limited Use)
我们严禁任何人员在未经许可的情况下接触用户数据，严禁将数据用于广告投放、信用评估或模型训练。
